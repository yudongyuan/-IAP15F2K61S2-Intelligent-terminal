# IAP15F2K61S2-Intelligent-terminal
·编写 DS18B20 单总线驱动，设计温度补偿测距算法，测距精度稳定在 ±3cm 以内，校准值可动态调整

·构建循环缓冲区，每 50ms 采样一次距离值，通过按键触发，期间屏蔽其他按键操作以保证数据完整性 

·编写I²C驱动控制PCF8591 DAC模块，按原始采样间隔将缓冲区数据还原为模拟电压输出，实现波形回放 

·通过UART读取缓冲区距离数据，编写Python脚本使用matplotlib绘制距离-时间曲线，完成波形回放验证 

·利用SPI接口扩展外部 Flash 存储芯片，实现多组记录数据的掉电保存与读取，提升数据可靠性

·通过优化定时器中断，采用状态机消抖，最终测距数据刷新率稳定在≤1s，按键响应时间≤0.1s
